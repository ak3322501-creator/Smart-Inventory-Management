# Smart Inventory Management System Report

## Project Objective
Build a smart inventory management system that supports product tracking, supplier management, stock updates, sales recording, low-stock alerts, and predictive reorder suggestions.

## Development Methodology
- Individual project using AI-assisted design and implementation
- Agile-style incremental development with separate backend and frontend modules
- GitHub repository tracking and documentation included

## Key Features
- Product inventory dashboard
- Supplier directory
- Sales record entry and stock decrement
- Low stock monitoring
- 30-day average sales prediction for reorder planning

## Tools and Technologies
- Frontend: React, Vite
- Backend: Express, SQLite, Node.js
- AI Tools Used: ChatGPT / GitHub Copilot for code suggestions and architecture guidance
- Deployment plan: local dev server, with future deployment to Vercel/Railway
