# Smart Inventory Management System

A full-stack smart inventory management system for tracking products, suppliers, stock updates, low-stock alerts, sales records, and predictive inventory forecasting.

## Features

- Product catalog with stock tracking and inventory updates
- Supplier management
- Sales recording with stock decrement
- Low-stock alerts
- Predictive inventory suggestions based on recent sales
- React frontend + Express backend + local JSON data store

## Project Structure

- `backend/` - Express API with local JSON persistence
- `frontend/` - React UI built with Vite
- `docs/` - design and architecture documentation
- `report/` - project report and methodology brief
- `testing/` - test plan and verification notes

## Get Started

### Root setup (single command)

```bash
cd "d:/software eng"
npm install
node backend/init-db.js
npm start
```

### Backend

```bash
cd "d:/software eng/backend"
npm install
node init-db.js
npm start
```

### Frontend

```bash
cd "d:/software eng/frontend"
npm install
npm run dev
```
